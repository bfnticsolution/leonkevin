Questions fréquentes

Comment créer un compte WariPay?

Téléchargez l'application depuis le Play Store, ouvrez-la et cliquez sur "Créer un compte". Vous devrez fournir votre numéro de téléphone et créer un code PIN sécurisé. La création prend moins d'une minute.

Quels sont les frais pour les transferts?

Les frais varient selon le montant et la destination du transfert. Les transferts locaux sont sans frais et internationaux sont des frais compétitifs à partir de 0.5%. Vous pouvez voir les frais exacts dans l'application avant de confirmer chaque transaction.

Comment récupérer mon mot de passe oublié?

Cliquez sur "Assistance" à l'écran de connexion. Nous enverrons un code de vérification à votre numéro enregistré. Après vérification, vous pourrez créer un nouveau mot de passe sécurisé.

WariPay est-il sécurisé?

Absolument! Nous utilisons un cryptage bancaire de niveau militaire, une authentification à deux facteurs et une surveillance 24/7 pour protéger votre argent et vos données. Vous pouvez également activer la vérification biométrique pour plus de sécurité.
